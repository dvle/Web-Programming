MoneyUp

MoneyUP is a financial literacy and budgeting application designed to help college students manage their finances more effectively. Many students struggle with overspending and lack the tools to anticipate future financial outcomes.
MoneyUP addresses this challenge by combining education, budgeting, transaction tracking, and predictive insights into a single platform. It provides real-time feedback and forecasts to support more informed, proactive financial decisions.
The goal of the project is to improve financial awareness, reduce financial stress, and help students build long-term healthy money habits.

  • Framework & Languages: Flutter, Dart
  • Backend: Supabase, Plaid
